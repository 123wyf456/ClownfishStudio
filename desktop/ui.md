开发一个 Windows AI 电台桌面客户端。

技术栈：
- Electron
- React
- TailwindCSS
- Framer Motion
- shadcn/ui

目标：
实现一个“高级 AI 陪伴电台”UI，
重点是高级感和工业设计感。

风格要求：
- neo-skeuomorphic
- Teenage Engineering 风格
- Braun 风格
- Arc Browser 风格
- 极简工业设计
- 高级硬件设备感
- 米白色背景
- 大圆角
- 柔和阴影
- 极少颜色
- 强留白
- 不像网页
- 不像后台管理系统

禁止：
- 不要 dashboard 风格
- 不要默认 HTML input/button/select
- 不要截图贴图
- 不要 absolute 到处定位
- 不要复杂渐变
- 不要花哨配色
- 不要信息密度太高

视觉要求：
- 所有UI使用纯CSS/Tailwind实现
- 使用统一 design system
- 统一圆角
- 统一阴影
- 统一 spacing
- 使用 flex/grid 布局
- 使用组件化开发

页面结构：
1. 顶部：
黑胶播放器区域
左侧黑胶唱片
右侧播放信息

2. 中部：
天气 / 时间 / 音量旋钮

3. 下部：
电台推荐卡片

4. 底部：
AI聊天区域

要求：
- 当前阶段不要实现功能
- 不接API
- 不实现真实播放器
- 优先实现视觉质感
- UI必须接近工业产品
- 更像硬件设备
- 更像AI companion
而不是网页播放器
